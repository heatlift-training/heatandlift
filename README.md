# Heat & Lift Training

App web de análisis biomecánico de saltos para básquetbol.

## URL de la app
Una vez publicada en GitHub Pages, la URL será:
```
https://TU_USUARIO.github.io/heatandlift/
```

## Cómo subir actualizaciones
1. Ir al repositorio en github.com
2. Clic en `index.html` → ícono del lápiz (editar)
3. O usar **Add file → Upload files** para reemplazar el archivo
4. Commit → los cambios se publican en ~1 minuto

## Tests disponibles
- Squat Jump (SJ)
- Countermovement Jump (CMJ)
- Abalakov
- Drop Jump / RSI
- Salto en Distancia

## Sincronización con Google Sheets
Ver instrucciones en Ajustes dentro de la app.
